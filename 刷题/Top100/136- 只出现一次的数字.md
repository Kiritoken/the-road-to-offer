#  只出现一次的数字

## 题目描述
    给定一个非空整数数组，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。

    说明：

    你的算法应该具有线性时间复杂度。 你可以不使用额外空间来实现吗？

    示例 1:
    输入: [2,2,1]
    输出: 1

    示例 2:
    输入: [4,1,2,1,2]
    输出: 4

## 题目难度
    Tags： 简单

## 题目解析：
+ 使用位运算异或，遇到这种类型的题目，先想想位运算的可行性
+ 相同元素异或为0
+ 0和任何数异或都等于那个数本身

```Java
class Solution {
    public int singleNumber(int[] nums) {
        int res = 0;
        for(int i = 0; i< nums.length;i++) {
            res ^= nums[i];
        }
        return res;
    }
}
```
